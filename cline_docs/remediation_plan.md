# Website Remediation Plan
Based on the audit report dated 2/6/2025

## High Priority Tasks

### 1. Execute Link Building Strategy (Links Grade: F)
- Current Status: Weak level of backlink activity (76 backlinks)
- Action Items:
  * Create high-quality, linkable content
  * Reach out to local Bakersfield businesses and organizations
  * Submit site to relevant gaming and eSports directories
  * Create profiles on gaming industry websites
  * Develop relationships with gaming news sites and bloggers
  * Consider guest posting opportunities

## Medium Priority Tasks

### 2. Implement XML Sitemap (SEO)
- Create comprehensive XML sitemap
- Submit to search engines
- Include in robots.txt file
- Set up regular sitemap updates

### 3. Implement robots.txt File (SEO)
- Create robots.txt file at root level
- Include XML sitemap location
- Define crawl rules for search engines
- Block access to admin and private areas

### 4. Increase Title Tag Length (SEO)
- Current: "Bakersfield eSports Center" (26 chars)
- Target: 50-60 chars with key services/location
- Update all main pages with optimized titles

### 5. Add Canonical Tags (SEO)
- Add canonical tags to all pages
- Prevent duplicate content issues
- Specify preferred URL versions

## Low Priority Tasks

### 6. Content & Technical Improvements
- Update link URLs to be more human-readable
- Increase page text content (target: 500+ words)
- Optimize images (reduce file sizes)
- Remove inline styles (move to CSS)
- Add schema markup for better search visibility
- Add X Cards for better social sharing
- Implement Facebook Pixel for tracking

### 7. Mobile Optimization
- Current Mobile PageSpeed: 57
- Target Score: 85+
- Action Items:

  * Minimize render-blocking resources
  * Reduce redirect chains
  * Improve load time metrics
