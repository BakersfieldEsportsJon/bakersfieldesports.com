# Technical Context

## Technologies
- HTML5/CSS3/JavaScript for frontend
- PHP 7.4+ for backend functionality
- Node.js for build tools and optimization

## Development Tools
1. Build & Optimization:
   - clean-css-cli for CSS minification
   - Performance monitoring script
   - Browser developer tools

2. Performance Monitoring:
   - Navigation Timing API
   - Performance API
   - Resource Timing API
   - Beacon API for analytics

## Development Setup
1. CSS Processing:
   ```bash
   npm run minify-css    # Minifies CSS files
   ```

2. Performance Monitoring:
   - View metrics in browser console
   - Production metrics sent to /analytics

## Technical Constraints
- Browser compatibility requirements
- Performance budget targets
- Analytics endpoint configuration needed
- Resource loading optimization
