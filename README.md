# Bakersfield eSports Center Website

This repository contains the website and admin dashboard for Bakersfield eSports Center.

See `STYLEGUIDE.md` for guidelines on working with CSS and brand assets.
Run `npm run lint:css` to check formatting with Stylelint.
Legacy helpers in `css/custom.css`, `css/layout.css`, and `css/utilities.css`
contain TODO notes for remaining lint issues.
